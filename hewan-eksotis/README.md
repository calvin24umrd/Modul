Cara Menjalankan Proyek
Unduh atau clone seluruh folder proyek ini ke komputer Anda.

Pastikan struktur folder sudah sesuai dengan skema di atas.

Klik kanan pada file index.html dan pilih Open with (Buka dengan), lalu pilih browser web favorit Anda (seperti Google Chrome, Mozilla Firefox, atau Microsoft Edge).

Pastikan perangkat Anda terhubung ke internet agar file Bootstrap dan Ikon dari CDN dapat dimuat dengan sempurna.

Penjelasan Kode Singkat
index.html: Dibagi menjadi beberapa <section> semantik seperti #dashboard, #katalog, dan #tambah-data untuk memudahkan pembacaan kode dan navigasi anchor link dari navbar.

style.css: Berisi penyesuaian yang tidak disediakan secara bawaan oleh Bootstrap, seperti gambar latar belakang pada Hero Section dengan efek gradient overlay, transisi smooth scrolling, dan animasi hover pada kartu produk dan dasbor.
