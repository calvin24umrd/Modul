Program Sistem Manajemen Sepatu ini merupakan aplikasi berbasis web sederhana yang dibuat menggunakan HTML sebagai struktur tampilan, CSS untuk desain dan styling, serta JavaScript untuk mengatur interaksi dan logika program. Pada bagian tampilan, website ini memiliki navbar dengan fitur Dark Mode yang tersimpan menggunakan localStorage sehingga tema tetap aktif meskipun halaman di-refresh. Selain itu, terdapat dashboard ringkas yang menampilkan informasi jumlah stok, kategori, dan total produk. Di bagian utama terdapat daftar produk sepatu yang masing-masing memiliki tombol “Beli” dan “Wishlist”, di mana ketika tombol beli ditekan maka stok akan otomatis berkurang dan tidak bisa dibeli lagi jika stok habis, sedangkan tombol wishlist akan menambahkan produk ke dalam daftar wishlist yang ditampilkan melalui modal popup dengan jumlah item yang dihitung secara otomatis. Program ini juga memanfaatkan Bootstrap untuk membuat tampilan lebih responsif dan rapi. Secara keseluruhan, aplikasi ini dirancang untuk mencakup sistem manajemen penjualan sepatu sederhana yang interaktif dan ramah pengguna serta dapat dikembangkan lebih lanjut menjadi sistem yang lebih kompleks.
